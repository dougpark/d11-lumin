# UI Enhancement
- make the UI more user-friendly and visually appealing

# Main Message Page - Remove bottom Nav bar
- remove the bottom navigation bar from the message page to provide more screen space for messages.
- this is often called "Slide-in Page Transition" or "Stack-based routing."
- add a "<" "Back" button to the top left of the page to navigate to the Channels page.
- add a Search magnifying glass icon to the top right of the page to navigate to the Search page.
- add a Person icon to the top right of the page to navigate to the Profile page.
- remove the Mentions button all together from the message page, as it is not needed with the new design.

# Channel List Page
- make a full slide-in panel from the left side of the screen that shows the list of channels.
- make the message page slide in from the right side of the screen when a channel is selected.
- add a new Header bar to the top of the Channel List page that shows "Lumin Chat" with icon
- add a new button to go to the Lumin Dashboard page (https://d11.me) 
- add the user name to the top of the Channel List page.


# v3 ideas:

Other Strong UI Design Improvements

Add unread badges per channel in the channel list, not just global new message badge.
Add channel pinning and recent channels section for faster navigation.
Add subtle haptic-like visual feedback on mobile taps using short transform/opacity transitions.
Improve composer with attachment affordance and persistent draft per channel.
Add inline date jump shortcuts in long histories, for example Today, Yesterday, This week.
Add message grouping by sender/time window to reduce visual noise.
Add swipe gestures: swipe right from messages to return to channels.
Add accessibility pass: larger touch targets, stronger contrast on muted text, visible focus rings.
Add empty-state design for no channels and no messages with clear primary actions.
Add optimistic skeleton placeholders for channel list and messages during initial load.
